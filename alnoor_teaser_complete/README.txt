AL NOOR TEASER WEBSITE

HOW TO USE:

1. Upload the ENTIRE folder to GitHub.
2. Make sure index.html stays in the main folder.
3. Keep the sounds folder beside it.

Folder structure:

index.html
sounds/
  ambient.wav
  boom.wav
  hit.wav
  rise.wav
  click.wav

Enable GitHub Pages:
Settings → Pages → Deploy from branch → main → /(root)

Your site will work automatically.
